Represent the server.

Fields

Name Description
val name: String

The server's name.
var commands: Array<Command>

The list of commands available. IMPORTANT: Commands must not have duplicates names. If two commands have the same name, an error will occur.
val timeout: Long

Optional: Timeout in milliseconds before AsyncCommands return an timeout result. Default value: 15000 = 15 seconds.
val securityConfiguration: SecurityConfiguration

Optional: Represent how the server must manage security. Default value: No security.
