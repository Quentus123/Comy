Project Name

Comy

Comy is a library that make server creation easier, a mobile application is provided with the server.

Features

  •  No need to create an user interface thanks to Comy iOS, an application designated for your users !
  •  Create commands that can be trigerred from users.
  •  Let users to pass parameters to commands
  •  User authentification

In the future

  •  Create a chat server with channels, etc...
  •  Send notifications to users
  •  Create permissions groups of users (Admins, Members, ...)

Availability

Beta version of Comy will be released soon (next week probably), feel free to message me for any question!

Mobile App

Open Source iOS app is currently under devloppement, you can see code here : https://github.com/Quentus123/Comy-iOS
